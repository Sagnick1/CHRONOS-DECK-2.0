# The Chronos-Deck 2.0

A "GitHub Pages Ready" Student Operating System built with pure HTML, CSS, and Vanilla JavaScript.

## Features

- **Cyberpunk Theme**: Neon visuals, CRT scanlines, and glitch effects.
- **System Boot Sequence**: Simulated loading animation with "kernel mount" effects.
- **AI Oracle**: Integrated Google Gemini API with a terminal-style chat interface.
- **Focus Hub**: Zen mode timer with visual countdown and audio feedback.
- **Study Planner**: LocalStorage-based To-Do list and analytics charts.
- **Zero Build Tools**: No Node.js, Webpack, or complex setup required.

## Deployment

This project is designed to be deployed instantly to **GitHub Pages**.

1. Create a new repository on GitHub.
2. Upload `index.html`, `style.css`, and `script.js`.
3. Go to **Settings** > **Pages**.
4. Select the `main` branch and save.
5. Your OS is live!

## Usage

- **Default PIN**: `0000` (or set your own on first login).
- **AI Usage**: Type your query in the Oracle terminal and press Enter.
- **Zen Mode**: Toggle in the Focus Hub to hide distractions.
